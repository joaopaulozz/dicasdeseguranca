Backups: importância e melhores práticas



-O que é Backups?

Backup é um termo utilizado para uma atividade que consiste em realizar cópias de segurança de dados digitais de um dispositivo, como fotos, documentos, softwares ou qualquer arquivo digital, com o intuito de recuperá-los em caso de perdas.

-As vantagens do backup em nuvem:

  1. – Acesso remoto;
  2. – Informações criptografadas;
  3. – Investimento menor que o tradicional;
  4. – Profissional especializado;
  5. – Escalabilidade;
  6. – Agilidade;
  7. – Segurança.

-Tipos de backup:

  1. Backup completo: faz uma cópia de todos os arquivos que você tem no computador;
  2. Backup incremental: Esse tipo de backup serve para fazer cópias apenas dos arquivos que foram alterados ou criados do zero após o backup normal;
  3. Backup diferencial: É baseado no backup de dados completo anterior mais recente, captura apenas os dados que foram alterados desde o backup completo;
  4. Backup diário: Copia todos os arquivos criados ou modificados na data atual, no mesmo dia em que ele é feito, e também não sinaliza os arquivos que passaram pelo backup. ;

-Quais são as melhores práticas de backup?

  1. .Escolha a forma de armazenamento e o local de backup;
  2. Defina o que vai ser armazenado;
  3. Mantenha uma rotina de segurança;
